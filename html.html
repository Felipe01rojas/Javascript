¿QUÉ ES HTML?

“HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata

de la siglas que corresponden a HyperText Markup Language, es decir,

Lenguaje de Marcas de Hipertexto”.

Esto lo hace por medio de las marcas de hipertexto las cuales son etiquetas

conocidas en inglés como tags.

¿Cómo funciona un editor HTML?

Un editor HTML es un software para editar y crear código HTML que se utiliza para sitios web

u otros documentos web. Con los editores HTML basados en texto, el código fuente puede ser

editado directamente. y es un lenguaje estándar que los navegadores, son capaces de leer y reproducir.
